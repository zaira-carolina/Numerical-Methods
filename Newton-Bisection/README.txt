lib_fun: contiene las funciones de tarea (comentadas), en dado caso que se quiera 
explorar otra función, hay que añadirla.

lib_metodos: contiene los dos métodos en funciones y la función de derivada que 
se utilizó para el método de Newton

biseccion_newton.cpp : es el archivo a ejecutar, en el main() tiene sólo
llamadas al usuario para escoger la cantidad de iteraciones, el método
o los métodos que se van a usar, el intervalo para el método de Bisección
y la aproximación inicial para el método de Newton


**** ¿Cómo ejecutar? ***
// Puede escribirse este comando en Windows o Linux 
g++ biseccion_newton.cpp
